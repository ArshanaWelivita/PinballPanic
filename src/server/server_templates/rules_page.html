Game Rules

Gameplay Overview

The objective of the game is to guide the ball from the entry point to the exit point while avoiding obstacles.

  • Each level features a grid with various objects.
  • You must input the correct coordinates for the ball's destination.
  • Objects like bumpers and tunnels can affect the ball's path.

How to Play

Enter the coordinates for the destination cell and submit your answer. If correct, you progress to the next level!
